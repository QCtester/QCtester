vi--常用命令

菜鸟/小白初识vi及它的升级版本vim。
哈哈哈，其实认识还不多，所以先分享一下几个常用的vi/vim编辑器的命令。

  • 进入基本模式/command
  • 进入插入模式/Insert
  • 退出插入模式
  • 视图模式/Visual
  • 退出视图模式
  • 保存当前编辑，但不退出。使用w命令时，可以再给编辑文件起一个新的文件名。
  • 保存当前内容到指定newfile中，而原有文件保持不变。

若newfile是一个已存在的文件，则给出提示信息：
File exists(use!to override)

  • 不保存退出

一般会给出提示：
No write since last change(use ! to overrides)

  • 保存并退出

以上只是一小部分比较常用的命令，掌握了就基本能做一些修改保存的操作了。但vi/vim的命令还有很多的，大家可以借助神奇的互联网，继续深入学习哈～！

以下是我使用vi时遇到的一些错误/提示：

  • 打开Vi编辑器出现E325: ATTENTION，如：⬇️⬇️⬇️
  1. 按回车 进入vi编辑器；
  2. 输入:q! 退出vi编辑器；
  3. 在终端输入rm -f /etc/.profile.swp 然后按回车；
  4. 再次进入vi编辑器，之前的提示就消失了。
