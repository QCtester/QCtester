pip--安装教程

参照加自创：http://blog.csdn.net/leorx01/article/details/60956565

配置pycharm时候参考 配置puthon版本：http://www.jianshu.com/p/96bcc261b75a

1.安装pip
1.1
https://pypi.python.org/pypi/pip
pip9.0.1 选择‘Installation’选项跳转到安装文档指引界面，按照提示下载get-pip.py文件

1.2.python get-pip.py
如果出现permisson denyed ：sudo chown -R Qing/Library/Python/2.7

2.安装selenium：pip3 install selenium

3.下载firefox
http://www.firefox.com.cn/download/

4.下载pycharm
https://www.jetbrains.com/pycharm/download/#section=mac

出现引用selenium问题 ，在编译器中点击提示‘install selenium’ 自己安装的。不晓得是自己安装，还是从自己的篇python环境中拷贝了一份过去的。
